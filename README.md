# RockyVi

RockyLinux bootc images with some UBlue utilities.


This uses code & workflows from [FinPilot](https://github.com/castrojo/finpilot) and [Bluefin LTS](https://github.com/ublue-os/bluefin-lts) both licensed under Apache 2.0


Included tools:
- Gnome (Rocky version, might switch to hyperscale)
- Tailscale
- Docker/Podman/Qemu/KVM
- Brew.sh
- ProjectBluefin's commons
- Iwd
